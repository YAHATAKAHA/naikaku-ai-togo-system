# Naikaku AI Togo System v0.1.0

This is a public-source developer preview from 合同会社EMYSTI.

## What This Release Is

- A React/Vite TypeScript workbench for a governed multi-model AI cabinet.
- A local, auditable engineering loop that can split one mission into cabinet roles, runner handoffs, receipts, evidence review, and artifact audit.
- A bring-your-own-provider system for OpenAI-compatible providers, OpenRouter, Anthropic, Aliyun DashScope/Qwen, Gemini, local endpoints, and custom gateways.
- A macOS-friendly developer preview archive for local evaluation.

## Quick Start

```bash
tar -xzf naikaku-ai-togo-system-0.1.0-mac-dev-preview.tar.gz
cd naikaku-ai-togo-system-0.1.0-mac-dev-preview
npm ci
npm run dev
```

For gateway-backed provider and runner features:

```bash
npm run gateway
```
