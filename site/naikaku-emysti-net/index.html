AI の仕事に、内閣制の統合を。

Naikaku AI 統合システムは、計画、コード実行、サンドボックス、 批評、監督、証拠付き反復をひとつの運用画面にまとめる、 日本発の多モデル AI 内閣制ワークベンチです。

日本語を主言語に、英語・中国語にも対応。ユーザー自身の API キーでも、 将来の EMYSTI API Gateway でも動く設計を目指しています。

自分の API でも、将来の EMYSTI Gateway でも。

自分の API キーを使う

  • 役割ごとに provider、endpoint、model、API-key alias を設定。
  • ローカル gateway 経由でセッション単位の secret を扱う設計。
  • 費用を使わずに配線確認できる local replay provider。

EMYSTI API Gateway を使う

  • ルーティング、課金、監査、モデル fallback の公式入口として計画。
  • ユーザー所有の API キーを置き換えるのではなく、隣に置く設計。
  • 複数 provider 契約やコスト管理を簡単にしたいチーム向け。
