v0.1.0 · macOS Developer Preview

Mac で試せる公開プレビュー。

現在の配布物は署名済みの `.dmg` ではなく、Node.js 22 以上で動かす開発者向けアーカイブです。 技術検証、非商用評価、貢献準備に使えます。 

tar -xzf naikaku-ai-togo-system-0.1.0-mac-dev-preview.tar.gz
cd naikaku-ai-togo-system-0.1.0-mac-dev-preview
npm ci
npm run dev

Download

GitHub Release から取得できます。

tar.gz download zip download checksums SHA-256 release notes GitHub

必要なもの

macOS、Node.js 22 以上、npm。外部 runner は任意です。

無料検証

`npm run open-source:mvp-check` で有料 API キーなしの導線検証ができます。

Native Mac app

署名済み `.dmg`、自動更新、権限チェックは次段階のパッケージング課題です。
