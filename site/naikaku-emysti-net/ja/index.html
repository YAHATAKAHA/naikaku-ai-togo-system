AI の仕事に、内閣制の統合を。

Naikaku AI 統合システムは、計画、コード実行、サンドボックス、批評、 監督、証拠付き反復をひとつの運用画面にまとめる、日本発の AI ワークベンチです。

v0.1.0 をダウンロード GitHub を見る

ひとつの依頼を、複数の役割で監督し、証拠付きで進める。

単一の AI に丸投げするのではなく、首相、批評、監督、実行、安全、記憶、採点の役割を分けます。 実行はプリセット、準備確認、実行記録、証拠レビュー、成果物監査の後ろに置きます。

内閣制ガバナンス

提案する AI、実行する AI、批評する AI、監督する AI を分け、各ループで判断を残します。

自分の API を使える

OpenAI 互換、OpenRouter、Anthropic、Aliyun DashScope/Qwen、Gemini、ローカルモデルに対応する設計です。

Mac とローカル実行を重視

Codex CLI、Claude Code、OpenHands、OpenClaw 系ツール、Hammerspoon などを監督付き runner として接続する方向です。

目的から選ぶ。

まず試す、API をつなぐ、Mac で runner を動かす、貢献する。 それぞれの入口を分け、必要な情報へまっすぐ進めるようにしています。

よくある質問

Naikaku AI 統合システムは何をするものですか？

複数の AI 役割が計画、実行、批評、監督、採点を分担し、証拠付きで作業を進める AI 内閣制ワークベンチです。

自分の API キーを使えますか？

はい。役割ごとに provider、endpoint、model、API-key alias を設定する設計です。

商用利用できますか？

公開版は非商用利用向けです。商用利用、SaaS、顧客納品、商用改造には合同会社EMYSTI の書面許諾が必要です。
